## Run Locally

Clone the project

```bash
  git clone https://github.com/ChiragRohada7020/Turf.git
```

Go to the project directory

```bash
  cd frontend
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
